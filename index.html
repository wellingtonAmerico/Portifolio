Linguagens de marcação e estilo, utilizadas para a criação do Front-End, páginas estáticas, no desenvolvimento Web. (o que você vê nesta e em todas as outras paginas da internet 😉).

Último elemento da tríade do desenvolvimento Front-End, mas também pode ser utilizada no Back-End (lado do servidor). Esta linguagem de programação que dá vida a página, além de processar e transformar dados enviados e recebidos. Ela que é responsável pelas funcionalidades dinâmicas dos sites, como autocompletes e movimentos. Como exemplos, temos meu nome "sendo escrito" e este menu lateral.

Esta linguagem de script é aplicada em automatizações da página assim como o Javascript e o Python, no entanto é mais utilizada no Back-End. Esta linguagem de programação é uma das mais utilizadas no mundo.

Parâmetro para trabalhar com Bancos de Dados relacionais. Utilizando o SQL que criamos um Banco de dados e manipulamos os dados nele contido.

A linguagem de programação C é retratada como a linguagem mãe, por diversas outras linguagens surgirem à utilizando como base. Tanto o C quanto o C++ são programação orientado a objeto, paradigmas de desenvolvimentomuito utilizado atualmente.

Extremamente versátil, podendo ser utilizada para construir aplicativos, fazer análise de dados, desenvolver programas, inteligência artificial, criar jogos, e muitas outras atividades. Uma das linguagens de programação mais utilizadas no mundo, assim como o PHP.
